# Campus Facilities Reservation System

## 📌 Description
A web-based reservation system developed to help users find available hostel or campus rooms and make secure bookings through an organized online platform.

## 🚀 Features
- User registration and login
- Search available hostel/campus rooms
- Book rooms online
- View booking history
- Manage user profile
- Secure reservation process
- Database-driven room and booking management

## 🧰 Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL


## ⚙️ Installation & Setup
1. Clone the repository
2. Place the project folder inside your local server directory (`htdocs` for XAMPP)
3. Import the MySQL database using:
   - `hostel_booking_schema.sql`
4. Configure database connection in `db.php`
5. Start Apache and MySQL
6. Open the project in your browser

## 📂 Main Modules
- Authentication
- Room search and availability
- Booking management
- Booking history
- User profile management
